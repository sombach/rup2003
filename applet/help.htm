Tips on Using the Search Engine

The search engine allows you to search for pages in the Rational Unified Process (RUP) and Rational Developer Network (RDN) in a number of ways. For example, you can:

  • search for pages that contain any of the words that you have typed.
  • search for pages that contain all of the words that you have typed.
  • Search for pages that contain the exact phrase that you have typed.
  • Search for pages that contain none of the words that you have typed.

To enter a search query, type the words to be searched for in the Any word, All the words, Exact phrase, and/or Without the word fields, and then press ENTER or click the Search Now button. When the search is complete, the title and summary of each matching page appear in the Results field. Click a title to open the page in your RUP Web browser window. If you have performed a RDN search, the results appear in a separate RDN Web browser window.

For example, to search for pages that contain either or both of the words "adopt" and "vision," and all of the words "Rational," "Unified," and "Process," type the words adopt vision in the Any word field and Rational Unified Process in the All the words field.

You can select how many results per page that you want by using the Show list. If the results are more than what you selected to see per page, click the next and previous buttons to page through the results.

You can also indicate whether you want the query to be applied against RUP or RDN. To choose between RUP and RDN, click the In section list, and then select the desired section.

Finding a Word on a Page

Once a page is displayed by the search engine, use the Web browser's search tool to find a specific word on that page. Press CTRL+F to start the Web browser's search tool.

Entering a Search Query

A search query consists of one or more specified words. Boolean operators cannot be used. Instead of Boolean operators, use the Any word, All the words, Exact phrase, or Without the word fields that are provided. The search process is not case-sensitive, which means that Hello, HELLO, and hElLo are all considered the same. The wildcard symbol * is not supported.

When more than one field is used, the query is evaluated with precedence from top to bottom. For example, the query:

Any word:              adopt vision

All the words:        project management

Exact phrase:        Rational Unified Process

Without the word: implementation

is evaluated as (adopt OR vision) AND (project AND management) AND (Rational Unified Process) NOT (implementation). In other words, the matching pages must contain the words "adopt" or "vision," and all of "project" and "management," along with the name "Rational Unified Process." They must not contain the word "implementation."

Special Considerations

  • The search engine automatically excludes common words such as "where," "when," and "the," because they tend to slow down your search without improving the precision of the results.
  • Boolean operators cannot be used. See the section titled Entering a Search Query for instructions on how to enter a search query instead.
  • The wildcard * is not supported.
  • In order for a query in the Without the word field to make sense, there must be text in at least one of the other search fields. In other words, the search engine cannot find pages that do not have certain words unless the you have specified that those pages must contain other words or phrases.
  • To search for results in the RDN section, you must have a RDN account in order to view the results. If you do not have an account or cannot be automatically logged in, you are redirected to the RDN log-on screen.
